# Script de Conceito de UI para Stitch

PRD — Stitch
Acquisition Intelligence Platform (Lead Operations OS)

Objetivo deste documento:
Orientar o Stitch a criar todas as telas, fluxos visuais e microcopies do produto, respeitando a natureza do sistema como uma Plataforma de Estratégia, Orquestração e Inteligência de Captação de Leads, não um CRM tradicional.

1. CONTEXTO DO PRODUTO (PARA O STITCH)
1.1 O que este produto É

Um Control Plane de Aquisição

Um Sistema Operacional de Leads

Uma Plataforma de Estratégia + Orquestração + Inteligência

Multi-projeto / Multi-conta

IA-first, Event-driven, Strategy-first

1.2 O que este produto NÃO é

❌ Um CRM tradicional

❌ Um funil linear simples

❌ Um gerenciador de contatos

❌ Uma ferramenta de disparo em massa

Princípio-chave:
Leads são consequência de estratégia, não o ponto de partida.
